Navbar 만들어보기

부트스트랩 CSS로 네비게이션 바 만들기 대부분의 부트스트랩 UI 컴포넌트들은 문서로 정리가 잘 되있는 편 !
But, 네비게이션 바의 경우 좀 더 자세한 설명이 필요 !
네비게이션 바는 그리드와 큰 연관이 있고, 부트스트랩에서 가장 중요하고 , 자주 사용되는 UI 컴포넌트이기 때문

네비게이션 바의 컨텐츠 / 구조와 정렬
네비게이션 바에서는 부트스트랩이 지원하는 컨텐츠만을 사용해야함
- 로고 / 브랜드
- 네비게이션 링크 / 버튼 / 드롭다운 버튼
- 연락 규격
- 텍스트
네비게이션 바 내에 – 그리드 기반 행과 열 규격을 따로 사용 하지 말자
네비게이션 바 내부에는 네비게이션을 위한 자체 규격에 대한 제한이 있다.
예를 들어 상단 예제 ml-auto (좌측 자동마진 처리) 같은 클래스는 두번째 네비게이션 요소를 우측으로 정렬하기 위해 사용되었다.
Navbar
  • Left aligned
  • Right alinged
위의 내용을 입력 하면 이런 Navbar가 생성되어야 한다.
그렇지 않다면 css , js 를 제대로 세팅했는지 확인해보자 !
