# 사전작업

- git 설치하기
git은 일종의 버전 관리툴
자신의 PC버전에 맞는 녀석으로 설치합니다.
설치방법이 어렵다면 010-5882-4......

- 작업 디렉토리 만들기
터미널과 조금 친해지는 시간을 가질겸, 바탕화면에 likelion 폴더를 만들어봅시다. 바탕화면에서 마우스 오른쪽 클릭 후 git bash here를 통해 터미널을 키고,
아래 명령을 내려봅시다.
mkdir likelion

완성되 었으면 likelion 폴더 안으로 들어가봅시다.
cd likelion

- python 설치확인하기

파이썬이 잘 설치되어 있나 확인해 봅시다. 터미널창에 아래 명령어를 입력합니다.
python --version

설치된 파이썬 버전이 제대로 출력되면 설치가 성공된겁니다. 설치했는데 안뜬다면 path 설정이 잘못됬거나, 설치를 제대로 안한거니 다시 확인해보세요.

# VS Code 셋팅하기


git도 설치했고, 파이썬도 설치해서 정상작동되는걸 확인 했다면 이제 VS Code를 본격적으로 작동 시킬 차례입니다. 터미널의 경로가 앞서 만든 likelion 폴더 안에 있는 상태에서 아래 명령어를 입력합니다.
~/Desktop/likelion 이라고 쓰인 노란 글씨가 현재 위치입니다.


code .
vs code를 실행하는 명령입니다. 여기서 `.`은 현재 디렉토리를 열어라 라는 의미입니다.

명령어를 치면 이제 VS code가 likelion 폴더를 기준으로 열립니다. (VS code를 켜고 `파일 → 폴더열기`를 실행한 것과 동일합니다.) VS code에서 작업을 하면서 터미널도 다룰 수 있습니다. 단축키 `ctrl` + `shift` + ` 를 눌러보세요. 만나서 반가우니까 파이썬이 설치됬는지 한번 더 버전을 확인해봅시다. # VENV 생성 & 가상환경 실행
(왜하는지 묻지도 따지지도 말고 그냥 하시면 되는 부분입니다. 가상환경 설정이라고 주절주절 설명하려면 말은 길어지는데 Django개발을 위해서 항상 맨 먼저 해야하는 것 이라고 생각하면 됩니다.
git도 설치한 여러분들은 이것도 해낼 수 있습니다.)

## python -m venv 가상환경명
python -m venv myvenv 위 명령을 치고 나면 옆에 폴더가 하나 생깁니다.
`myvenv`라는 가상환경명을 사용해 가상환경을 만들었습니다.
가상환경명은 아무거나 할수 있지만 보통 `myvenv`나 `venv`같은 네이밍을 사용합니다. 복잡하게 지으면 명령어 칠때 여러분만 힘듭니다.
뭔지 모르는 저 폴더가 가상환경입니다. 가상환경을 만들었으니 사용해 봅시다.

## 가상환경 실행하기

source myvenv/Scripts/activate

위 명령어를 치면 터미널에 뭔가 바뀝니다. (`source` 대신 `.` 을 입력해도 됩니다.)
명령어 줄위에 `(myvenv)` 이런게 생겼습니다.
가상환경이 실행되었다는 뜻입니다.

> **Django개발을 할때는 무조건! 가상환경부터 키고 시작한다.**

기억해 둡시다. 중요한 부분입니다. 가상환경이 뭔가요? 궁금하다면 구글에 물어보길 바랍니다.

## 가상환경 끄기
켰으면 끌 줄도 알아야하니까 알려드립니다.

deactivate

모르겠다 싶으면 그냥 터미널을 껐다 켜도 됩니다..

# Django 설치하기
이제 드디어 장고를 설치할 차례입니다. (**먼저, 꼭, 가상환경을 실행합니다!)**
아래 명령어를 입력하세요.

pip install django

pip는 파이썬관련 패키지(누가 만들어놓은 파이썬 프로그램들)를 설치하기 위해 사용되는 명령어입니다.
install은 설치하라는 거고, django를 설치하라는 뜻입니다.
- 특정 버전 장고 설치하기

pip install django==3.0.8

`==`기호를 이용해서 설치하고 싶은 장고버전을 설치할 수 있습니다. 이부분을 안쓰면 그냥 배포중인 최신버전 장고가 설치됩니다.
명령어를 치면 설치가 막 진행됩니다.
설치가 다 진행된 모습입니다. 주황색 글씨로 pip 버전이 구리니까 업그레이드를 해라는 얘기가 있는데...
해주도록 합시다. 저기 python -m 어쩌구 부분을 터미널에 입력만 해주면 됩니다. 장고까지 설치했다면 이제 프로젝트를 진행할 준비가 막 되었습니다. 축하합니다. Django까지 설치를 마쳤습니다. {%endblock%}
